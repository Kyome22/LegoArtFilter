# LegoArtFilterDemo

Demonstration of LegoArtFilter for iOS/macOS.<br/>
This project runs on both iOS (14≤) and macOS (11≤).

## Libraries

- [LegoColors](https://github.com/Kyome22/LegoColors)
- [LegoArtFilter](https://github.com/Kyome22/LegoArtFilter)
